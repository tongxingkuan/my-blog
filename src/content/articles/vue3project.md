## vue3项目开发

### 安装方式

1. `nuxt3` 引入方式
```cmd
npx nuxi init <project-name>
```
2. `vite` 引入方式

```cmd
npm init vite-app <project-name>
```

3. 文件引入

```html
<script src='vue.global.js'></script>
```

4. 包管理工具引入

```cmd
npm create vue@latest
```
